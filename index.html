Common Goals

Technology can be used to create software to improve our way of living. From automating and simplifying manual task.

intercommunication

Writing software like anything else is done in colaboration with others. As such communcation is key. In an environment where many developers are working together they need to be able to quickly and efficiently commuicate together. It can be cumbersome to leave ones desk and find the needed person and email is not on par with other communcation tools of today like instant messaging.

However, in a proffesional environment with many people one large group chat would not do. Thats where Slack comes in. It allows for instant messaging with people inside an organization. Inside an organization there are channels (chat rooms) to specific topic to provide organized conversation.

Information and Feedback

Github is one of the tools, developers use to work together in collaberation. It's allows for people working on the same project to push (upload) their contrubtions and pull (download) others additions. It's sorta like a shared drive like DropBox or Google Drive but for code.
